# Formal R package named triangdist that implements the Triangular Distribution. Assignment from Probability and Distribution (2025-2026).

## Elements:
• dtriang(x, min, max, mode): Density function.
• ptriang(q, min, max, mode): Distribution function.
• qtriang(p, min, max, mode): Quantile function.
• rtriang(n, min, max, mode): Random generation.

## Installing:
To install the package, run in the terminal the following command:
```
install.packages("remotes")
remotes::install_github("aceva/triangdist")
```
